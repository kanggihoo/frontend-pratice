# 📚 모던 가격표(Modern Pricing Table) 실습 가이드

## [주제 및 기획 의도]

이번 프로젝트의 주제는 B2B SaaS 혹은 다양한 서비스의 랜딩 페이지에서 흔히 볼 수 있는 **'모던 가격표(Modern Pricing Table)'**입니다.
단순해보일 수 있지만 간격(Gap), 그리드(Grid) 정렬, 그리고 메인 상품을 돋보이게 하는 **시각적 강조(Visual Hierarchy)**까지, 실무에서 굉장히 자주 다루게 되는 레이아웃 요소들을 골고루 경험해 볼 수 있도록 기획되었습니다.

---

## [학습 목표]

- **Grid와 Flexbox의 실전 조합**: 전체 큰 틀은 CSS Grid를 통해 반응형으로 분기하고, 내부의 세세한 요소 정렬(가격 텍스트 베이스라인 맞추기, 아이콘과 텍스트 옆으로 배치하기 등)은 Flexbox를 적절히 믹스하여 사용하는 방법을 익힙니다.
- **반응형 웹 UI 구현**: 모바일에서는 카드들이 위아래로 쌓이고(`grid-cols-1`), 데스크톱 화면(`md:`)에서는 3개의 카드가 나란히 배치되는(`md:grid-cols-3`) 기법을 경험합니다.
- **시각적 강조 효과 디자인**: 단순히 색상을 다르게 하는 것을 넘어, 카드 자체가 살짝 튀어나온 듯한 입체감(`-translate-y-4`)과 그림자(`shadow-2xl`)를 통해 사용자의 시선을 '프로 플랜'으로 유도해봅니다.

---

## [교육자의 철학 / 강조 포인트]

_"가장 잘 팔아야 하는 상품을 돋보이게 만드는 것이 UI의 역할입니다."_
제가 이번 레이아웃을 구성하며 가장 강조하고 싶었던 것은 중앙 **"프로" 카드의 처리 방식**입니다.
다른 카드들과 차별화를 두기 위해 어두운 톤(`bg-indigo-900`)을 적용하였고, 무엇보다 `absolute` 위치와 `-translate-y-1/2` transform을 이용해 카드의 테두리 선분에 걸치도록 **'Most Popular' 뱃지**를 멋지게 배치했습니다. 현업에서 정말 많이 쓰이는 팁이니 꼭 클래스 구조를 눈여겨봐주세요!

---

## [핵심 학습 개념]

`completed.html`에서 집중적으로 살펴보시면 좋을 Tailwind CSS의 핵심 유틸리티들은 다음과 같습니다:

1. **반응형 Grid**: `grid grid-cols-1 md:grid-cols-3 gap-8`
   화면 크기에 따라 열의 개수를 손쉽게 조절하는 방법입니다.
2. **요소 위치 세밀 조정 (Transform)**: `md:-translate-y-4`, `transform -translate-y-1/2`
   Y축 방향으로 요소를 끌어올려 입체감이나 요소 간 레이어 오버랩 효과를 구현합니다.
3. **절대 위치 지정 (Positioning)**: `relative`와 `absolute top-0 right-8`
   부모 요소를 기준으로 자식 요소(여기서는 뱃지)를 내가 원하는 정확한 좌표에 띄우는 기법입니다.
4. **그라디언트 및 텍스트 효과**: `bg-gradient-to-r from-pink-500 to-orange-400`, `tracking-tight`
   트렌디하고 생동감 있는 버튼/뱃지 배경을 만들고 자간을 조절해 모던한 타이포그래피를 완성합니다.

---

## [최종 기대 효과]

이 과제를 끝마치고 나면, 여러분은 **여백(Padding/Margin)의 스케일링, 테두리 반경(Rounded) 적용, 그림자(Shadow)를 활용한 입체감 형성**에 깊은 자신감을 갖게 될 것입니다. 즉, 어떤 복잡한 랜딩 페이지의 컴포넌트를 보더라도 당황하지 않고 '이건 Grid로 잡고, 내부는 Flex로 묶으면 되겠구나' 하는 **UI 아키텍처를 설계하는 눈(Takeaway)**을 얻게 됩니다.

## [추천 실습 순서]

가장 효과적인 학습을 위해 `practice.html`을 열고 아래의 순서대로 클래스를 채워나가 보세요.

1. **전체 뼈대와 헤더 잡기**: `<body>`의 배경색과 중앙 정렬 클래스를 작성하고, 요금제 안내 제목 텍스트의 크기와 여백을 설정합니다.
2. **그리드 컨테이너 활성화**: `<div class="grid...">` 구조를 만들고 모바일과 데스크톱 화면에서 카드가 어떻게 나뉘는지 브라우저 창을 줄였다 늘렸다 하며 확인합니다.
3. **양쪽 베이직/엔터프라이즈 카드 디자인**: 흰색 배경, 테두리, 그림자, 그리고 내부 패딩을 줘서 카드의 형태를 만듭니다. 버튼의 Hover 애니메이션도 시도해봅니다.
4. **중심이 되는 '프로' 카드 부각시키기**: 배경색을 어둡게 적용하고, 글자색을 하얗게 반전시킨 뒤 카드를 위로 살짝 들어올리는(`translate`) 효과를 줍니다.
5. **디테일 완성 (Most Popular 뱃지)**:카드의 상단을 기준으로 뱃지가 어떻게 절대위치(`absolute`)로 자리를 잡는지 HTML 주석을 보며 완성합니다.
