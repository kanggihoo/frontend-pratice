# 팀 프로필 카드 섹션 - Tailwind CSS 실습 #2

## 주제 및 기획 의도

이번 실습은 **팀 소개 프로필 카드 섹션**입니다. 스타트업이나 에이전시 웹사이트의 "Our Team" 페이지에서 흔히 볼 수 있는 UI 패턴으로, 어두운 배경 위에 **글래스모피즘(Glassmorphism)** 스타일의 카드를 배치한 모던 디자인입니다.

이전 실습(가격표)이 밝은 배경 + 단순 Grid 레이아웃이었다면, 이번에는 **다크 테마 + 그라디언트 배경 + 반투명 카드 + 인터랙티브 호버 효과**로 완전히 다른 디자인 언어를 경험합니다.

---

## 학습 목표

이 실습을 통해 다음 능력을 기르게 됩니다:

1. **다크 테마 UI 구성력**: 밝은 배경과는 전혀 다른 색상 체계(투명도, 밝기 조절)를 이해합니다.
2. **글래스모피즘 구현력**: `backdrop-blur`, `bg-white/5` 같은 투명도 유틸리티를 자유자재로 사용합니다.
3. **group hover 패턴 이해**: 부모 요소에 마우스를 올렸을 때 자식 요소가 반응하는 패턴을 체득합니다.
4. **그라디언트 텍스트 기법**: `bg-clip-text`와 `text-transparent`를 조합한 고급 텍스트 스타일링을 익힙니다.
5. **반응형 Grid 마스터**: 1열 → 2열 → 4열로 변화하는 다단계 반응형 레이아웃을 구성합니다.
6. **absolute 포지셔닝 활용**: 이미지 위에 오버레이, 뱃지 등을 정확하게 배치하는 방법을 익힙니다.

## 교육자의 철학 / 강조 포인트

### "투명도(Opacity)는 다크 테마의 생명이다"

밝은 배경에서는 `bg-gray-100`, `border-gray-200`처럼 **색상 자체**로 계층을 구분합니다. 하지만 다크 테마에서는 **투명도**로 계층을 만듭니다.

```
bg-white/5   → 가장 뒤 레이어 (거의 안 보임)
bg-white/10  → 호버 시 살짝 밝아짐
bg-white/20  → 더 강조된 요소
border-white/10 → 은은한 경계선
```

이 투명도 시스템을 이해하면, 어떤 다크 테마든 자연스럽게 구현할 수 있습니다.

### "group은 Tailwind의 숨은 무기다"

`group`과 `group-hover:`의 조합은 JavaScript 없이도 풍부한 인터랙션을 만들 수 있는 강력한 도구입니다. 카드에 마우스를 올렸을 때 내부 이미지가 확대되는 효과는 이 패턴의 대표적인 활용 사례입니다.

```html
<div class="group">                          <!-- 부모: group 선언 -->
  <img class="group-hover:scale-110" />      <!-- 자식: 부모 hover 시 반응 -->
</div>
```

## 핵심 학습 개념

### 1. 다방향 그라디언트 (`bg-gradient-to-br`)
- `to-br` (bottom-right), `to-t` (top), `to-r` (right) 등 방향 지정
- `from-` → `via-` → `to-` 3단계 색상 지정으로 풍부한 그라디언트

### 2. 글래스모피즘 효과
- `backdrop-blur-sm/md`: 배경을 흐리게 처리
- `bg-white/5`: 매우 낮은 불투명도로 반투명 느낌
- `border border-white/10`: 유리 느낌의 은은한 테두리

### 3. Group Hover 패턴
- 부모에 `group` 클래스 → 자식에 `group-hover:` 접두사
- `group-hover:scale-110`: 부모 hover 시 자식 확대
- 카드 전체에 마우스를 올리면 이미지만 확대되는 효과

### 4. 그라디언트 텍스트 (`bg-clip-text`)
```
bg-gradient-to-r   → 그라디언트 배경 설정
from-purple-400     → 시작 색상
to-pink-400         → 끝 색상
bg-clip-text        → 배경을 텍스트 모양으로 자르기
text-transparent    → 텍스트 색을 투명하게 (배경이 보이도록)
```

### 5. 이미지 오버레이 기법
- `relative` 컨테이너 안에 `absolute inset-0`으로 오버레이 배치
- `bg-gradient-to-t from-slate-900/90 to-transparent`: 하단이 자연스럽게 어두워지는 효과

### 6. 반응형 Grid 분기
```
grid-cols-1       → 모바일: 1열
sm:grid-cols-2    → 태블릿(640px~): 2열
lg:grid-cols-4    → 데스크탑(1024px~): 4열
```

### 7. 색상 그림자 (`shadow-{color}`)
- `hover:shadow-2xl hover:shadow-purple-500/20`: 그림자에 색상을 입혀 네온 느낌
- 카드마다 다른 색상 그림자로 개성 표현

### 8. 애니메이션 유틸리티
- `animate-pulse`: 깜빡이는 효과 (온라인 상태 표시)
- `transition-all duration-500`: 모든 속성 부드럽게 전환
- `hover:-translate-y-2`: 호버 시 위로 살짝 떠오르는 효과

## 최종 기대 효과

이 과제를 완료하면 다음을 자신의 무기로 만들 수 있습니다:

1. **다크 테마 디자인 시스템**: 투명도 기반의 계층 구조를 자유롭게 설계할 수 있습니다.
2. **글래스모피즘 UI**: 모던 웹에서 유행하는 반투명 UI를 Tailwind만으로 구현할 수 있습니다.
3. **인터랙티브 카드 패턴**: `group`, `transition`, `transform`을 조합한 호버 효과를 자유자재로 적용할 수 있습니다.
4. **그라디언트 활용**: 배경, 텍스트, 버튼, 오버레이 등 다양한 곳에 그라디언트를 적용하는 기법을 익힙니다.
5. **절대 위치 레이어링**: 이미지 위에 뱃지, 오버레이 등을 정확하게 배치하는 능력을 갖춥니다.
