당신은 뛰어난 프론트엔드 웹 개발자이자, 초보자의 눈높이에 맞춰 친절하게 지식을 전달하는 'Tailwind CSS 교육 전문가'입니다.

저의 목적은 완성된 UI 디자인을 눈으로 확인한 뒤, 스타일이 전혀 적용되지 않은 뼈대 HTML에서 시작하여 직접 Tailwind CSS 클래스를 추가해보며 실전 감각을 기르는 것입니다.

이 학습을 위해 다음 **3가지 형태의 결과물**을 작성하여 제공해 주세요. (구현할 UI 테마는 랜딩 페이지의 예쁜 프로필 카드, 가격표, 혹은 대시보드 위젯 등 임의로 하나를 정해서 멋지게 만들어주세요.)

**1. 완성본 HTML (`completed.html`)**

- Tailwind CSS가 완벽하게 적용된 HTML 파일입니다.
- `<script src="https://cdn.tailwindcss.com"></script>`를 포함하여 별도 설치 없이 바로 브라우저에서 실행 가능하게 해주세요.
- 모던하고 깔끔한 디자인(색상, 여백, 그림자, 반응형 등)이 적용되어 있어야 합니다.
- 이때 tailwindCss 4.x 버전으로 작성해주세요.

**2. 실습용 뼈대 HTML (`practice.html`)**

- `completed.html`과 완벽히 동일한 DOM(HTML 태그) 구조를 가집니다.
- **하지만 Tailwind CSS의 유틸리티 클래스는 전부 비워두거나 제거해주세요.**
- 단, 사용자가 이 코드만 보고도 무엇을 채워 넣어야 할지 알 수 있도록 구역마다 **아주 상세하고 친절한 HTML 주석**을 달아주세요.
- 주석 작성 예시: `<!-- [프로필 이미지 래퍼] 이곳은 Flex 컨테이너로 만들고, 요소를 중앙에 배치하세요. 배경은 둥글게, 약간의 그림자와 테두리 색상을 옅게 추가하면 좋습니다. -->`

**3. 학습 가이드 문서 (`README.md`)**
교육자의 입장에서 다음의 내용들이 깊이 있게 포함된 가이드를 작성해주세요.

- **[주제 및 기획 의도]**: 이번에 디자인한 컴포넌트(예: 랜딩 페이지 내비게이션, 갤러리 카드 등)가 무엇인지 소개해주세요.
- **[학습 목표]**: 이 결과물이 구체적으로 **어떤 능력을 기르기 위해**, 혹은 **어떤 것을 경험해 보게 하기 위해** 설계되었는지 알려주세요.
- **[교육자의 철학 / 강조 포인트]**: 실제 코드를 작성한 교육자의 입장에서, 해당 레이아웃을 구성할 때 '이것만은 꼭 알아야 한다'고 가장 강조하고 싶었던 부분이 무엇인지 설명해주세요.
- **[핵심 학습 개념]**: `completed.html`에 작성된 코드를 통해 Tailwind CSS의 어떤 핵심 기능들(예: Flexbox/Grid 활용법, 여백 스케일링, 반응형 분기 처리 등)을 배우게 되는지 짚어주세요.
- **[최종 기대 효과]**: 이 과제를 끝마치고 났을 때, 학습자가 코드를 통해 명확히 깨닫거나 자신의 무기로 만들었으면 하는 요소(Takeaway)가 무엇인지 작성해 주세요.
- **[추천 실습 순서]**: 사용자가 이 과제를 어떻게 단계별로 진행해야 가장 효과적인지 행동 가이드라인을 제시해 주세요.

결과물은 각각의 파일 단위로 코드 블록에 담아 명확히 구분해서 출력해 주세요.
